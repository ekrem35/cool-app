# cool-app
React native code challange project

*Basic auth with placeholder api*

### The project uses these

- TypeScript
- Native Base UI
- Redux
- Router Flux

# To use

`git clone https://github.com/ekrem35/cool-app.git`

`cd cool-app`

`npm i`

`npx react-native run-android`
